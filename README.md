# 🎓 Student Score Predictor (ML Project)

## 📌 Overview
This project predicts student exam scores based on study hours using Machine Learning.

It uses **Linear Regression** to learn the relationship between study time and performance.

---

## 🚀 Features
- 📊 Predicts score based on study hours  
- 🎯 Interactive web app using Streamlit  
- 📈 Graph visualization of predictions  
- 💾 Model saving using joblib  
- 🌐 Deployed online  

---

## 🛠️ Tech Stack
- Python  
- pandas  
- scikit-learn  
- matplotlib  
- Streamlit  

---

## 📊 How It Works
1. Data is loaded from CSV file  
2. Model is trained using Linear Regression  
3. User inputs study hours  
4. Model predicts the score  
5. Graph shows relationship between hours and score
